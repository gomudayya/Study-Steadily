### Flex 사용

- FlexBox에 Flex 속성을 이용해서 비율 기반으로 UI를 작성

### MaxWidth, MaxHeight, MinWidth, MinHeight, Width, Height

- Max~ , Min~ 을 사용해서 상한선, 하한선을 %로 걸어두고, width, height를 사용

### Dimensions API

- 실제 기기의 치수를 가져옴.
	- Screen 사이즈 - 상태바 포함 
	- Window 사이즈 - 상태바 미포함
- 가져온 치수를 통해서 Style 변경
