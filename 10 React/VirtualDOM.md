
- 리액트에서 컴포넌트 함수가 재실행된다고 해서, DOM에 다시 재 삽입 되는것이 아니다.
- 리액트는 Virtual DOM을 사용해서 성능을 최적화 한다.
	- 리액트 컴포넌트 함수가 재실행
	- 최신의 Virtual DOM 생성
	- 이전 버전의 Virtual DOM과 비교를 통해 달라진 부분을 체크함.
	- 달라진 부분을 실제 DOM 트리에 업데이트
- Virtual DOM은 실제 DOM보다 훨씬 가볍다.
	- 순수 자바스크립트 객체이다.
	- 실제 DOM은 스타일 레이아웃 이벤트 등을 모두 반영한 복잡하고, 무거운 구조이다.
